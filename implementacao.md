# Implementação

Para o desenvolvimento do projeto, iniciou-se com o menu principal, desenvolvendo a troca de páginas com as opções e créditos do jogo.
Então foram adicionadas as opções de múltiplas resoluções e troca de idioma no jogo.

Com o menu construído, foi iniciado o desenvolvimento do jogo em si, com a criação da cena, utilizando a leitura de arquivos JSON como forma de permitir a troca entre ambientes.
Com a cena, foi incluso a construção do fundo e cálculos para a conversão entre o mundo isométrico e as posições vistas pela câmera.

Para a funcionalidade, foi adicionado o personagem, sua movimentação e colisão com objetos. Após esta construção, foi adicionada as interações.
Foi desenvolvido para o projeto a interação de diálogo, onde ao se aproximar, um prompt aparece e ao pressionar o botão, a caixa de diálogo aparece.
O código foi deixado preparado para receber os outros métodos de interação, como a adição de itens no inventário e a troca de cenas, mas estes não foram implementados.

Para a construção do projeto, além do uso do QT, foi utilizada para o projeto: Dungeon Scrawl para a construção básica do fundo; arte de AxulArt para o modelo de personagem usado de teste; fonte de Pixel Sagas; objetos de user interface de Kenney; e arte de Mark Gosbell como objetos de cenário.
